# Wedding Dani & Marini

Undangan pernikahan digital untuk M. Daniansyah Chusyaidin, S.Kom & Siti Nur Marini, A.Md.M — Sabtu, 29 Agustus 2026, Surabaya.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```
   npm install
   ```
2. Run the app:
   ```
   npm run dev
   ```
3. Open `http://localhost:3000`

## Guest Personalization

Add `?to=` query parameter to personalize the invitation:

```
http://localhost:3000/?to=Budi%20Santoso
```

## Build for Production

```
npm run build
npm run preview
```
